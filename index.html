Nokogiri-Mart

Store items from Wallmart.com using Nocogiri and Bootstrap

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Nokogiri-Mart.

Screen-scraping is a popular subject, it involves downloading a web page’s HTML and parsing it for data. We will fetch the price, image, image ALT attribute and product name from Walmart.com.

We are going to use Nokogiri for Screen-scraping, which can parse HTML and XML documents and extract content from them. Nokogiri is fast, it feels like we’re working more directly on the HTML document rather than doing everything through a DSL.

Features

  • Nokogiri
  • SelectorGadget
  • Bootstrap
